STUDENT RESULT MANAGEMENT SYSTEM - README
This project is a beginner-friendly Python application used to manage student academic records,
calculate results automatically,
and generate performance summaries.
Overview
The Student Result Management System is a console-based Python application that allows adding
student details, entering marks,
calculating total, percentage, grade, and viewing results.
Features
• Add New Student
• Enter Marks (with validation)
• Automatic result calculation
• View individual student result
• View all students summary
• Grade assignment
• Pass/Fail status
Grading Logic
• 90–100 → A+
• 80–89 → A
• 70–79 → B+
• 60–69 → B
• 50–59 → C
• Below 50 → F
Pass Criteria: Percentage ≥ 50%
Technologies Used
• Python 3
• No external libraries
